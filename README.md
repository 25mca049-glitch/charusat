# charusat

A collection of small projects and demos maintained together. This repository currently contains the `funfacts` Vite + React TypeScript app.

Contents
- `funfacts/` — a Vite + React + TypeScript demo application.

Getting started

Prerequisites:
- Node.js 18+ or compatible LTS
- npm or yarn

Install dependencies (from repository root):

```bash
npm install
```

Run the `funfacts` app:

```bash
cd funfacts
npm run dev
```

Build the `funfacts` app:

```bash
cd funfacts
npm run build
```
